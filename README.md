# CloudBeats — SoundCloud Player for Minecraft

Listen to SoundCloud without leaving your game. CloudBeats brings a full-featured music player into Minecraft with a clean in-game HUD and a dedicated search screen.

## Features

- Search tracks by name or paste any SoundCloud track / playlist URL
- No API key or account required — works out of the box
- Play local MP3 files from your PC
- Track queue with shuffle and repeat modes (off / all / one)
- Seek by clicking the progress bar or jumping to an exact second
- Favorites — save SoundCloud tracks and local files
- In-game HUD with playback controls, 3 sizes (S/M/L), 9 anchor positions, draggable
- Compact HUD mode — just the track name and progress bar, stays out of your way
- Full keybind support: pause, next/prev track, seek ±10s, volume, toggle HUD
- Mod Menu integration

## Requirements

- Minecraft 1.21.11
- Fabric Loader ≥ 0.19.2
- Fabric API

## Building from source

```bash
git clone https://github.com/solanafl1pper/minecraftcloud.git
cd minecraftcloud
./gradlew build
```

The built jar will be in `build/libs/`.
